Welcome! This portfolio showcases end to end data science, analytics, and engineering projects. I specialize in transforming large, complex datasets into actionable predictive models and automated pipelines.

## Projects

### 1\. [Crime Forecasting Project](./crime-forecasting)

* **Pitch:** Built a predictive model to forecast crime trends using historical data (2020–2025).
* **Highlights:** Data cleaning (pandas), predictive modeling (XGBoost), REST API (FastAPI), and an interactive Power BI dashboard.
* **Deliverables:** Clean dataset, trained ML model (`.pkl`), API, and Power BI report (PDF and PNG exports).

### 2\. [Online Payment Fraud Detection (Big Data Analysis)](./online-payment-fraud-detection)

* **Pitch:** Developed a high-performance classification system to detect fraudulent transactions within a 6.3 million record dataset.
* **Highlights:** Imbalanced class handling, model comparison Logistic Regression vs. Random Forest, and computational cost benefit analysis.
* **Deliverables:** Analytical Notebook, comparative model performance report, and optimized classification pipeline.
